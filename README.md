# Team Task Manager

A professional, full-stack task management application built with **.NET Core Web API**, **CouchDB**, and **Angular**.

## 🚀 Features
- **Authentication**: Secure Login/Signup with JWT.
- **Role-Based Access**: Admin and Member roles for controlled project/task management.
- **Project Management**: Create and track multiple projects.
- **Task Tracking**: Create, assign, and update task statuses (To Do, In Progress, Done).
- **Dashboard**: Real-time stats including total tasks, completion rates, and **Overdue** task tracking.
- **Premium UI**: Modern glassmorphic design with a vibrant, high-end aesthetic.

## ⚙️ Tech Stack
- **Backend**: .NET 8 Web API
- **Frontend**: Angular 17+ (Standalone Components)
- **Database**: CouchDB (NoSQL)
- **Styling**: Vanilla CSS (Modern CSS Variables & Glassmorphism)

## 🛠️ Local Setup

### Prerequisites
- .NET 8 SDK
- Node.js & npm
- CouchDB installed and running locally

### Running the Backend
1. Navigate to the `Backend` folder.
2. Update `appsettings.json` with your CouchDB credentials.
3. Run `dotnet run`.

### Running the Frontend
1. Navigate to the `Frontend` folder.
2. Run `npm install`.
3. Run `npm start` or `ng serve`.
4. Open `http://localhost:4200`.
